# Hướng dẫn đóng góp

Cảm ơn bạn đã muốn đóng góp cho **Awesome Vietnam Dev**!

## Quy tắc thêm tài nguyên

1. **Chất lượng trên số lượng** — Chỉ thêm tài nguyên thực sự hữu ích, không thêm trang spam hoặc nội dung kém.
2. **Còn hoạt động** — Link phải còn hoạt động và được cập nhật.
3. **Liên quan đến dev Việt Nam** — Ưu tiên nội dung tiếng Việt hoặc dành riêng cho cộng đồng VN.
4. **Thêm vào đúng mục** — Đặt tài nguyên vào mục phù hợp nhất.

## Cách tạo Pull Request

1. Fork repo này
2. Tạo branch mới: `git checkout -b add-ten-tai-nguyen`
3. Thêm tài nguyên vào `README.md` theo format:
   ```
   - [Tên tài nguyên](https://link.com) - Mô tả ngắn gọn bằng tiếng Việt.
   ```
4. Commit: `git commit -m "Add: Tên tài nguyên"`
5. Push và mở Pull Request

## Format mô tả

- Viết mô tả ngắn gọn, rõ ràng (1 câu)
- Bắt đầu bằng chữ hoa
- Không kết thúc bằng dấu chấm (đã có ở cuối)
- Ưu tiên mô tả tiếng Việt

## Báo cáo link hỏng

Nếu tìm thấy link hỏng, mở [Issue](../../issues/new) với tiêu đề: `[Dead Link] Tên tài nguyên`.
